# AutoLicense

## 소개
### 개요
- [경찰청 교통민원24(이파인)](https://www.efine.go.kr/main/main.do) 에서 진행할 수 있는 운전면허증 진위여부 확인작업입니다.<br>
(홈페이지 리뉴얼 이후 업데이트가 되지 않았습니다.)
